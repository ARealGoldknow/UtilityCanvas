# Vocal Canvas Desktop App

This is the full downloadable desktop version of Vocal Canvas.

## Requirements

- macOS
- Python 3.10+
- Built-in macOS commands: `say`, `afconvert`, `afplay`

## Run

```bash
python3 vocal_canvas_desktop.py
```

## Features

- Styled desktop interface
- Voice selector with local macOS voices
- Speed control (80-400)
- Instant preview playback
- WAV export
