Q: Is it safe?

Short answer: yes, for normal use it is designed to be safe, but it is still a beta build so you should treat it like early software and avoid feeding it sensitive private text.

The desktop app is focused on local TextToSpeech workflows and does not require always-on accounts, hidden background services, or automatic downloads. You choose when audio is generated, and you choose when files are exported. That keeps the workflow predictable and creator-friendly.

For the website demo, your text is sent to the demo backend to generate the preview audio, so use it for sample lines instead of confidential content. If you want maximum privacy, use the downloadable app for production work. Also make sure you only download builds from the official project links and keep your version updated as beta fixes are released.
